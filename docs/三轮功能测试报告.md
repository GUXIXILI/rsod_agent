## 四、BUG详细说明

### BUG-001：Agent对话失败（LLM 502 Bad Gateway）
- **优先级**: P0
- **状态**: 未修复（外部依赖问题）
- **现象**: 发送消息后AI无回复，后端日志显示LLM API返回502
- **错误日志**:
  ```
  Agent 流式执行异常: <html>
  <head><title>502 Bad Gateway</title></head>
  <body><center><h1>502 Bad Gateway</h1></center>
  <hr><center>nginx</center></body>
  </html>
  ```
- **推断原因**: 外部LLM服务不可用或网络问题
- **影响范围**: 智能对话、Agent工具调用、记忆增强

### BUG-006：工具返回格式非JSON
- **优先级**: P1
- **状态**: 未修复
- **现象**: 前端控制台报错：`[tool_end] 解析失败: SyntaxError: Unexpected token '【'`
- **详细描述**: Agent工具返回的结果使用中文括号【】格式，前端期望JSON格式
- **影响**: 前端有降级处理（纯文本展示），不影响基本功能但显示不规范

### BUG-007：会话列表401错误
- **优先级**: P2
- **状态**: 待确认
- **现象**: 前端控制台报401错误，但API直接调用正常
- **推断原因**: 前端请求拦截器未正确携带Token，或会话列表请求时机问题
- **影响**: 前端无法显示会话列表

### BUG-008：Element Plus Radio废弃警告
- **优先级**: P2
- **状态**: 未修复
- **现象**: 控制台警告：`[el-radio] [API] label act as value is about to be deprecated in version 3.0.0, please use value instead.`
- **影响**: 功能正常，仅警告信息

### BUG-009：视频检测失败 - MinIO 431错误
- **优先级**: P1
- **状态**: 新发现
- **现象**: 视频上传后检测失败，状态为failed
- **错误日志**:
  ```
  视频异步任务失败: task_id=183, error=non-XML response from server;
  Response code: 431, Content-Type: text/plain; charset=utf-8,
  Body: 431 Request Header Fields Too Large
  ```
- **错误位置**: `app/services/detection_service.py:372` - `minio_client.upload_bytes()`
- **推断原因**: MinIO服务器配置的请求头大小限制过小，上传大文件时请求头超过限制。可能是MinIO配置问题或上传方式问题。
- **影响范围**: 视频检测功能完全不可用

### BUG-010：个人设置缺少更新个人资料API
- **优先级**: P2
- **状态**: 新发现
- **现象**: PUT /api/user/profile 返回405 Method Not Allowed
- **详细描述**: 用户API只有GET /profile和PUT /password，缺少更新个人资料（昵称、邮箱、电话等）的接口
- **影响**: 用户无法修改个人信息

6. **Element Plus版本适配**: 更新Radio组件使用value属性替代label

## 六、测试结论

第三轮测试共验证了36个测试用例，其中26个通过，5个失败，5个因外部依赖（LLM 502）受限。

**已修复验证通过的BUG（4个）**:
- ✅ BUG-002：数据看板页面空白
- ✅ BUG-003：检测结果annotated_url为null
- ✅ BUG-004：个人信息不显示
- ✅ BUG-005：RAG知识库搜索无结果

**仍存在的BUG（6个）**:
- ❌ BUG-001：Agent对话失败（LLM 502，外部依赖）
- ❌ BUG-006：工具返回格式非JSON
- ❌ BUG-007：会话列表401错误
- ❌ BUG-008：Element Plus Radio废弃警告
- ❌ BUG-009：视频检测MinIO 431错误（新发现）
- ❌ BUG-010：个人设置缺少更新资料API（新发现）

**整体评价**: 核心检测功能（单图、批量、ZIP）和数据看板、历史记录、RAG知识库等功能运行稳定。主要问题集中在Agent对话（受LLM服务影响）、视频检测（MinIO配置问题）和部分前端UI细节。
