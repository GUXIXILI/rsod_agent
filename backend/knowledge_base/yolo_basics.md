# YOLO 目标检测基础知识

## 一、YOLO 算法原理

### 1.1 核心思想
YOLO（You Only Look Once）是一种单阶段（one-stage）目标检测算法，将目标检测问题转化为回归问题。与两阶段检测器（如 Faster R-CNN）不同，YOLO 只需要对图像进行一次前向传播，即可同时预测目标的边界框和类别概率。

### 1.2 网格划分
YOLO 将输入图像划分为 S × S 的网格。每个网格单元负责预测中心点落在该网格内的目标物体。每个网格预测 B 个边界框（bounding box），每个边界框包含 5 个参数：中心坐标 (x, y)、宽度 w、高度 h，以及置信度分数。

### 1.3 置信度
置信度（Confidence）定义为：
- Confidence = Pr(Object) × IoU(truth, pred)
- 如果网格中不存在目标，置信度应为 0
- 如果网格中存在目标，置信度等于预测框与真实框的 IoU

### 1.4 边界框预测
每个边界框预测 5 个值：(x, y, w, h, confidence)：
- (x, y)：边界框中心相对于网格单元边界的偏移量
- (w, h)：边界框的宽高相对于整张图像的比例
- confidence：置信度分数

## 二、关键概念

### 2.1 IoU（交并比）
IoU（Intersection over Union）用于衡量预测框与真实框之间的重叠程度：
- IoU = 交集面积 / 并集面积
- 取值范围 [0, 1]，值越大表示重叠越多
- 通常设定 IoU 阈值（如 0.5）来判断检测是否成功

### 2.2 NMS（非极大值抑制）
NMS（Non-Maximum Suppression）用于去除冗余的检测框：
1. 按置信度对所有检测框进行排序
2. 选择置信度最高的框，保留它
3. 计算该框与其余框的 IoU
4. 移除 IoU 超过阈值（如 0.5）的框
5. 重复步骤 2-4，直到处理完所有框

### 2.3 mAP（平均精度均值）
mAP（mean Average Precision）是目标检测的核心评估指标：
- 首先计算每个类别的 AP（Average Precision），即 Precision-Recall 曲线下的面积
- 然后对所有类别的 AP 取平均值得到 mAP
- mAP@0.5：IoU 阈值为 0.5 时的 mAP
- mAP@0.5:0.95：IoU 阈值从 0.5 到 0.95（步长 0.05）的平均 mAP

## 三、YOLOv11 改进

### 3.1 模型架构改进
YOLOv11 在 YOLOv8 的基础上进行了多项优化：
- 引入 C3k2 模块替代 C2f 模块，减少参数量
- 采用更高效的特征金字塔网络（FPN）结构
- 优化了检测头设计，支持多尺度特征融合

### 3.2 训练策略优化
- 改进的数据增强策略（Mosaic、MixUp 等）
- 自适应锚框计算
- 更优的损失函数设计（CIoU Loss）
- 学习率预热和余弦退火调度

### 3.3 模型变体
YOLOv11 提供多种模型规模：
- YOLOv11n（Nano）：最小模型，适合边缘设备
- YOLOv11s（Small）：轻量级模型
- YOLOv11m（Medium）：中等规模
- YOLOv11l（Large）：大模型
- YOLOv11x（Extra Large）：最大模型，精度最高

## 四、火灾烟雾检测特点

### 4.1 火灾检测的特殊性
火灾烟雾检测相比通用目标检测有以下特点：
- 目标形状不规则：火焰和烟雾没有固定形状
- 尺度变化大：从微小火苗到熊熊大火
- 背景复杂：不同场景下的干扰因素多
- 实时性要求高：火灾检测需要快速响应

### 4.2 烟雾检测难点
- 半透明特性：烟雾具有半透明性，与背景混合
- 颜色多样性：从白色、灰色到黑色
- 扩散特性：烟雾会随时间和空间扩散
- 光照影响：不同光照条件下烟雾外观差异大

### 4.3 火焰检测难点
- 动态变化：火焰形状和颜色不断变化
- 类间相似性：与灯光、日落等有相似特征
- 遮挡问题：火焰可能被物体遮挡
- 多尺度：同一场景中可能存在不同大小的火焰
