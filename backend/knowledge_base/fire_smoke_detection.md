# 火灾烟雾检测知识

## 一、火灾烟雾检测挑战

### 1.1 场景多样性
火灾烟雾检测需要应对多种复杂场景：
- 室内场景：住宅、办公室、仓库、商场等
- 室外场景：森林、草原、工业区、城市街道
- 特殊场景：隧道、矿井、地下停车场
- 天气影响：晴天、阴天、雨天、雾天

### 1.2 光照条件变化
不同光照条件对检测效果影响显著：
- 白天强光：火焰特征可能被阳光掩盖
- 夜间低光：红外或热成像辅助检测
- 逆光场景：目标特征被压缩
- 人工光源干扰：灯光、车灯等可能导致误检

### 1.3 目标尺度变化
- 初期火灾：火焰和烟雾面积小，特征不明显
- 发展期火灾：火焰迅速扩大，烟雾增加
- 完全燃烧期：大面积火焰和浓烟
- 衰减期：火焰逐渐减小，残留烟雾

## 二、烟雾特征分析

### 2.1 视觉特征
烟雾的视觉特征包括：
- 颜色特征：通常为白色、灰色、深灰色或黑色
- 纹理特征：烟雾区域纹理模糊，边缘不清晰
- 运动特征：烟雾具有上升和扩散的运动模式
- 透明度特征：烟雾具有半透明特性，能透出背景

### 2.2 时空特征
- 时间连续性：烟雾在时间上具有连续性，不会突然出现或消失
- 空间扩散性：烟雾从源头向外扩散，面积逐渐增大
- 运动方向：烟雾主要向上运动，受气流影响可能偏移
- 频率特征：烟雾区域的闪烁频率与火焰不同

### 2.3 检测策略
- 多帧差分：利用连续帧之间的差异检测运动区域
- 颜色空间分析：在 HSV、YCbCr 等颜色空间中分析烟雾特征
- 小波变换：利用小波变换分析烟雾的频域特征
- 深度学习特征：使用 CNN 自动提取烟雾的高层语义特征

## 三、火焰特征分析

### 3.1 颜色特征
火焰的颜色特征是其最显著的视觉特征：
- 红色分量：火焰区域红色通道值较高
- 饱和度：火焰区域通常具有较高的饱和度
- 亮度变化：火焰亮度随燃烧状态变化
- 颜色分布：从焰心到外焰颜色从白/蓝渐变到红/橙

### 3.2 形态特征
- 尖角特征：火焰通常具有尖角形状
- 面积变化：火焰面积随时间波动
- 周长面积比：火焰轮廓的不规则性
- 重心变化：火焰重心随时间轻微移动

### 3.3 动态特征
- 闪烁频率：火焰的闪烁频率通常在 1-10 Hz
- 形态变化：火焰形态不断变化，但整体位置稳定
- 颜色脉动：火焰颜色随燃烧强度变化
- 运动模式：火焰向上运动，偶尔有水平偏移

## 四、数据增强策略

### 4.1 几何变换
- 随机翻转：水平翻转、垂直翻转
- 随机旋转：小角度旋转（±15°）
- 随机缩放：模拟不同距离的火灾
- 随机裁剪：增强模型对局部特征的感知

### 4.2 颜色变换
- 亮度调整：模拟不同光照条件
- 对比度调整：增强或减弱对比度
- 色调调整：模拟不同光源色温
- 饱和度调整：模拟不同色彩饱和度

### 4.3 混合增强
- Mosaic 增强：将四张图像拼接成一张
- MixUp 增强：将两张图像按比例混合
- CutOut 增强：随机遮挡部分区域
- Copy-Paste 增强：将火灾区域复制到其他背景

## 五、误报处理

### 5.1 常见误报源
- 类火物体：灯光、日落、红色物体、反光
- 类烟物体：雾气、云、灰尘、蒸汽
- 运动物体：车辆、行人、动物
- 场景变化：光照突变、相机抖动

### 5.2 误报抑制策略
- 多帧确认：连续多帧检测到目标才报警
- 时序分析：分析目标的时间序列特征
- 多模态融合：结合温度、烟雾传感器数据
- 置信度阈值：提高报警的置信度阈值
- 区域过滤：排除已知的误报区域（如路灯、烟囱）

### 5.3 报警确认机制
- 首次检测：记录时间和位置，不立即报警
- 持续检测：连续 N 帧检测到目标时进入确认阶段
- 确认报警：满足确认条件后触发报警
- 报警取消：目标消失后在一定时间内取消报警
